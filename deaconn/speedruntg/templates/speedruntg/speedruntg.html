What Is Speedrun: The Game?

We are an open-source video game created and maintained by The Modding Community. We are geared towards the speedrunning community and strive to provide modding tools for the community to utilize so that anybody can create flavors, game modes, and maps for speedrunning!

Discord Server & Website

The community for this game is hosted within The Modding Community. The following is TMC's Discord server if you're interested in joining the community and having real-time conversations with other talented individuals!
