What Is Deaconn?

Deaconn is a company dedicated to Christian Deacon's products and career.

Christian Deacon is a self-taught technical engineer interested in programming, networking, and general technical engineering. He enjoys learning, progressing, and working on many areas within the technology industry. This ranges from (D)DoS protection, to pen-testing, to web development/general tools, and eventually AI/robotic engineering.

NOTE - This website is still in development!

Experience

Christian has experience in many programming languages and also has experience in both networking and Linux.

In addition to programming, Christian also has experience in Linux/networking. This comes with the following experiences.

  • Linux networking/tunneling protocols including IPIP, GFE, and more.
  • Linux kernel networking paths including XDP (Express Data Path) and TC (Traffic Control).
  • Kernel-bypass libraries such as the DPDK.
  • Linux firewalls including NFTables, IPTables, and more.
  • Third-party firewalls (either based on Linux or Unix) such as pfSense, IPFire, VyOS, and more.
  • BGP (Border Gateway Protocol) tuning and routing.
  • Deploying and managing Anycast systems.
  • Running Docker, Docker Compose, and Docker Networking.

Christian also has experience managing a large/successful gaming community for over ten years along with support-related jobs. This has given him the following additional experiences.

  • Managing over fifty dedicated servers/VPSs running a combination of Linux/Windows for game and web servers (along with custom kernels/configurations).
  • Providing the best customer experience.
  • Communicating and helping other community/server owners.
  • Writing public/internal guides and knowledgebase articles (Christian loves writing).
  • Microsoft Azure and Office 365.

Projects

Excluding official Deaconn products, the following are Christian's top projects.

  • GFL (Games For Life) - A large/successful gaming community Christian founded and owned from January 25th, 2011 to July 13th, 2021 (over ten years!).
  • Browser.TF - A web-sided server browser for the game Team Fortress 2.
  • XDP Firewall - A stateless firewall that attaches to the Linux XDP path.
  • XDP Forwarding - A layer 3/4 forwarding program that attaches to the Linux XDP path for fast packet processing and utilizes source-port mapping.
  • Packet Batch - A packet generator that supports three versions including the DPDK, AF_XDP, and standard Linux AF_PACKETv3 sockets. Useful for network monitoring or pen-testing (DoS/DDoS).
