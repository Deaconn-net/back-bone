What Is LBG?

LBG (Laid Back Gaming) is an online gaming community which hosts unique game servers across multiple games such as Garry's Mod, Rust, Counter-Strike: Global Offensive, Team Fortress 2, and more.

LBG is a separate entity from Deaconn, but Deaconn provides LBG with game server/web machines and technical assistance because Christian is an owner within LBG. LBG is more of a hobby to Christian and does it for fun/love for gaming.

Game Servers

We are currently running many servers in Counter-Strike, Rust, Garry's Mod, and Team Fortress 2. For a list of all game servers, please join our Discord server which is linked under the "Discord Server" section on this page!
